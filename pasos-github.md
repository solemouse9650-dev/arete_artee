# Pasos para subir a GitHub

## 1. Crea tu repositorio en GitHub
- Ve a github.com
- Click "New"
- Name: arete-galeria-arte
- Public: YES
- Click "Create repository"

## 2. Ejecuta estos comandos (reemplaza TU-USUARIO)

```bash
git remote add origin https://github.com/TU-USUARIO/arete-galeria-arte.git
git branch -M main
git push -u origin main
```

## 3. Si te pide usuario y contraseña
- Usuario: tu email de GitHub
- Contraseña: usa un "Personal Access Token" (no tu password normal)

## 4. Verifica que esté subido
- Ve a tu repositorio en GitHub
- Deberías ver todos los archivos
